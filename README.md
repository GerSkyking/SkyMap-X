# SkyMap-X
Arma Reforger Mod SkyMap-X ATAK Variant (Arma 3 CTab)

# Keybinds (Open Alpha – Subject to Change)

> Note: Keybinds are primarily intended for testing and may change in future updates.
---
## Power & Core Functions

- **Ctrl + F1** → Power On / Start / Off / Reset  
- **L Ctrl + 1** → GPS Mode  
  *(Modes 2–4: Chat, Settings, Feed – planned)*  
- **L Ctrl + F2** → Center map on player  
- **L Ctrl + F3** → Center & Lock camera to player  
- **Ctrl + Tab** → Mini Mode (~40% screen)  
- **Shift + Tab** → Big Mode (~85% screen)
  
---

## Map Controls

- **Page Up / Page Down** → Zoom In / Out  
- **L Ctrl + Page Up / Page Down** → Adjust Brightness  
- **R Ctrl + Arrow Keys** → Pan Map
   
---

# Handheld Interaction Mode

The tablet can also be taken into the player's hands and operated directly.

## Default Interaction

- Hold **R (Inspect)** to access the interaction options.

When holding the tablet, the interaction menu allows:

- Powering the tablet On / Off  
- Adjusting Zoom  
- Changing Brightness  
- Switching UI Modes  

---

## Planned for Handheld Mode

- Map Panning via interaction system  

---

 Some features may behave differently depending on the active UI mode  
> (Handheld / Mini / Big).  
> The system is currently in **Open Alpha** and under active development.
